Битовые строки
Описать класс "Битовая строка" BitString для хранения 64 битных целых беззнаковых чисел, члены-данные класса – два целых беззнаковых числа (типа unsigned int). Одно из чисел хранит старшую часть 64 битового числа, а второе хранит младшую часть 64 битового числа.

При создании объектов этого класса определить конструктор (конструкторы):

с двумя параметрами, в этом случае первый параметр старшая часть, а второй параметр младшая часть 64 битного числа
с одним параметром, в этом случае задается только младшая часть 64 битового числа, а старшая равна 0
по умолчанию, в этом случае обе части равны 0
Класс BitString должен содержать следующие методы:

void Print() - печать 64 битового числа в десятичном виде
void ShiftLeft(unsigned offset) - сдвиг влево битового представления числа на offset бит
void ShiftRight(unsigned offset) - сдвиг вправо битового представления числа на offset бит
void NOT() - операция битовой инверсии
void AND(const BitString& b) - операция побитового И с числом b
void OR(const BitString& b) - операция побитового ИЛИ с числом b
void XOR(const BitString& b) - операция побитового исключающего ИЛИ с числом b
В методах ShiftLeft() и ShiftRight(), если offset больше или равен 64, то результатом сдвига является 0
